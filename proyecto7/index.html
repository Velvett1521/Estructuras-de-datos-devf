Proceso del algoritmo

Divide and Conquer aplicado:

  1. Dividir: Se divide el arreglo en dos mitades
  2. Conquistar: Se resuelve recursivamente cada mitad
  3. Combinar: Se comparan los maximos de ambas mitades
  4. Caso base: Si el arreglo tiene un elemento, ese es el maximo
